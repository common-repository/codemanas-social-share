A simple social plugin, built with flexibility in mind.

== Description ==
CodeManas Social Share is a very simple, lightweight, social sharing plugin, that will allow user to add sharing links to any post, page or custom post type.

**What to Show:**

The plugin has options to show either of the following Social Sharing options:

* Facebook
* Linkedin
* Twitter
* Google Plus
* Whatsapp
* Pinterest

as well as the order of what to show easily via the backend.

**How to Show :**

You can add the image in any post via the backend options,

* Before the content
* After the content
* Floating Left
* Inside the featured image

or anywhere in the post / post types itself via a shortcode.

**Choose Colors:**

By default the plugin comes with sharing icons with their original colors, you can, if you choose change them and choose a custom color

**Choose Size for Social Icon:**

You can also specify, the size of the social icons ie. you can choose to display small, medium or large size icons.

== Installation ==
Installation Instructions :
1. Unzip codemanas-social-share.zip and Upload codemanas-social-share folder to the /wp-content/plugins/ directory
2. Activate the plugin through the ‘Plugins’ menu in WordPress
3. You can then see the \"Share\" option menu added to the admin sidebar, you can then begin customizing
